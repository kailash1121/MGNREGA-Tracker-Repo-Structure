MGNREGA-Tracker (Mobile-Friendly)
---------------------------------
1) Create GitHub repo 'MGNREGA-Tracker'
2) Copy-paste frontend_flutterflow.json, backend_nodejs folder, database folder, README.txt
3) Deploy backend_nodejs on Render/Railway/Heroku
4) Run database SQL files in MySQL (employees.sql, work.sql, payments_notifications.sql)
5) Import frontend_flutterflow.json into FlutterFlow, update API base_url to deployed backend, connect Firebase
6) Test app: login, dashboard, work entry, payments, reports, notifications
